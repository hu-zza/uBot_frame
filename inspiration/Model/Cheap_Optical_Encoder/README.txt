# Summary

This is just a mockup of a cheap optical sensor board from Aliexpress.  You can find it using the search terms "j34 optocoupler lm393."

I posted this as a remix because I found this other part using the board.  I thought it would be nice to have a modeled part for designing systems.  This is mostly to scale, but I rounded a little ( ie:  14.05mm = 14mm) so add a little tolerance if you use it for a design
