# Summary

Remix of onmioji's Holy Hand Grenade.  Used Mixmesher to reduce the triangles a bit then used Tinkercad to hollow it and split it and added the lip.
Update 4/23 Added a second version of the top with a bit tighter tolerance for the lip.  First version is quite loose once I printed it out.
