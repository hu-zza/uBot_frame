# Summary

This is holder for chinese motor like this one 
http://ru.aliexpress.com/item/TT-Motor-Smart-Car-Robot-Gear-Motor-for-Arduino-Free-Shipping-Wholesale/32536140897.html

Its using swallowtail as connection, so use only ABS! Do not use PLA, because of bad connection between peieces.
Print two T-shaped peices.
It have top connection via 2 holes for M4 screw. 
Motor attaching with T-connectors with 2 M3x40 screw.

Это держатель мотора с ласточкиным хвостом. Мотор китайский, по типу такого 
http://ru.aliexpress.com/item/TT-Motor-Smart-Car-Robot-Gear-Motor-for-Arduino-Free-Shipping-Wholesale/32536140897.html
Используйте ABS, тогда соединение будет прочным.
К конструкции деталь крепится с помощью М4 болтов.
Т-крепежи крепятся к мотору посредством М3х40 болтов.
Распечатайте два Т крепежа
