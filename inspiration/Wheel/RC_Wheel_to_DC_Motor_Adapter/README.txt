# Summary

This adapter is designed to connect a DC motor to standard RC wheels with a hex connector.

It is based on the discontinued DFRobot product found here:
https://www.dfrobot.com/product-645.html

I have tightened the slot connecting to the DC motor. I have also uploaded the SolidWorks file in case anyone needs to modify it.
