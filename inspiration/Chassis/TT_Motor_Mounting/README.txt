# Summary

This project is a collection of parts to mount TT Motors to vehicles. The first thing I've built with it was my 4WD Car. 

The car frame is super easy to print. It consist only of 3 parts: 
2x MotorMounting_Bridge2_length100_4scewHoles.stl 
1x simpleBaseFrame.stl 

In this git project you can find the complete part list and the source code for the Arduino Uno. 
https://github.com/grimmpp/arduino-rc-transmitter/tree/master/4WD_RC_Car_AddeptDriverBoard

You can find the whole project on github: https://github.com/grimmpp/tt-motor-mounting
